学术论文

PDFNet

arXiv GitHub

CVPR 2026 基于深度完整性先验与细粒度Patch策略的高精度二分图像分割

刘先杰，傅可人，赵启军

针对高分辨率二分图像分割（DIS）中非扩散方法精度不足、扩散方法计算昂贵的困境，本文引入单目伪深度图作为结构先验。我们发现前景深度方差显著低于背景（深度完整性先验），并设计多模态融合架构以及Depth Integrity-Prior Loss增强结构感知。提出的PDFNet以<50%参数量在DIS-5K达到SOTA，并且推理速度相较于扩散模型快约5倍。

HiDe

arXiv GitHub

arXiv 2026 HiDe：通过层次化解耦重新思考高分辨率MLLM中的放大方法

刘先杰，胡一曼，邹一雄，等

针对MLLMs在高分辨率图像上表现不佳的问题，此前归因于感知不足并采用"放大"策略。通过层次化解耦分析发现真正瓶颈是背景干扰而非物体大小，据此提出无需训练的HiDe框架。利用Token级注意力解耦（TAD）定位关键区域，并通过布局保持解耦（LPD）构建紧凑表示。HiDe在V*Bench上达到SOTA（Qwen2.5-VL 7B：92.1），超越RL方法，内存消耗降75%。

MLLM 高分辨率 视觉语言

E-VAds

arXiv

arXiv 2026 E-VAds：面向MLLM的电商短视频理解基准

刘先杰，胡一曼，吴亮，等

针对电商短视频缺乏专用基准、现有模型难处理开放商业意图推理的问题，本文提出首个电商短视频理解基准E-VAds。该基准包含3,961条高质量视频与19,785个开放问答，覆盖感知与推理两大维度五大任务，涵盖基础感知、跨模态检测、营销逻辑、消费者洞察与合规审查。本文设计多智能体标注系统保障标注质量，并提出MG-GRPO多粒度奖励策略。E-VAds-R1仅需数百样本即实现商业意图推理109.2%性能提升。

DIS-SAM

arXiv GitHub

ICME 2025 推动Segment Anything Model实现高精度二分图像分割

刘先杰，傅可人，江瑶，赵启军

本文提出一种两阶段框架DIS-SAM，将SAM与IS-Net级联实现高精度二分图像分割（DIS），在保留SAM可提示交互特性的同时显著提升边界细节精度。引入参数正交化与GT数据增强策略增强泛化能力。在DIS-5K数据集上，相比原始SAM、HQ-SAM、Pi-SAM分别提升约8.5%、6.9%、3.7%的Max F-measure，并在零样本场景展现出良好泛化性。
